# Pronoia

## The Word

Pronoia is the opposite of paranoia. Where paranoia is the feeling that the world is conspiring against you, pronoia is the feeling that the world is conspiring in your favor.

## What This Means for SlateOS

Every design decision in SlateOS starts from the assumption that the system should be working for the user, not against them.

**Defaults help.** The system ships with opinionated defaults that make things work immediately. You don't configure your way to a working desktop — you start with one and customize from there.

**No extraction.** No telemetry, no analytics, no ads, no engagement metrics, no dark patterns. The software does what you asked it to do and nothing else.

**Respect attention.** Notifications are minimal and meaningful. Animations serve comprehension, not decoration. The system never interrupts you to sell you something.

**Respect hardware.** The system is built to be fast on the hardware it targets. No Electron, no web views for core UI, no layers of abstraction burning cycles to render a dock. Rust and Wayland, directly on the GPU.

**Respect competence.** The system assumes you are capable. Settings are exposed, not hidden. The terminal is first-class. When something goes wrong, you get a real error message, not a friendly cartoon.

**Repair over replace.** The system is designed to be understood, modified, and fixed. Small Rust crates with clear boundaries. Human-readable config files. Service configs you can edit with a text editor.
